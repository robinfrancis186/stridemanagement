# STRIDE Management System

STRIDE COE Agentic AI Workflow Management System designed for the K-DISC Social Enterprise & Inclusion Division.

## Tech Stack
- Frontend: React 18, Vite, TypeScript, Tailwind CSS, shadcn-ui
- Backend: Supabase (PostgreSQL 16, Edge Functions)
- AI Integration: Claude API (via Edge Functions)

## Getting Started

### Prerequisites
- Node.js & npm
- Supabase CLI (if running edge functions locally)

### Installation
```bash
npm install
```

### Running Locally
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```
